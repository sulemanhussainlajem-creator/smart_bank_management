# Contributing to Smart Bank Management

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository (if you are not a collaborator).
2. Clone the repository to your local machine.

```bash
git clone https://github.com/sulemanhussainlajem-creator/smart_bank_management.git
```

3. Open the project in IntelliJ IDEA.
4. Configure the required JDK, JavaFX SDK, and MySQL database.
5. Create a new branch for your feature or bug fix.

```bash
git checkout -b feature/your-feature-name
```

## Project Structure

```
Smart_bank/
├── frontend/
├── backend/
├── resources/
└── database/
```

- **frontend/** – JavaFX UI classes
- **backend/** – Business logic and models
- **resources/** – CSS, FXML, images, icons
- **database/** – SQL scripts and database configuration

## Coding Guidelines

- Follow Java naming conventions.
- Write clear and readable code.
- Add comments where necessary.
- Keep methods small and focused.
- Test your changes before pushing.

## Commit Messages

Use meaningful commit messages.

Examples:

```
Add customer registration feature
Fix login authentication bug
Update database schema
Improve dashboard UI
```

## Pull Requests

Before submitting a Pull Request:

- Make sure the project builds successfully.
- Resolve merge conflicts.
- Keep Pull Requests focused on a single feature or fix.
- Provide a clear description of your changes.

## Reporting Issues

If you find a bug:

- Describe the problem clearly.
- Include steps to reproduce it.
- Attach screenshots if applicable.
- Mention your operating system and Java version.

## Code of Conduct

Be respectful and collaborative.

- Be constructive when reviewing code.
- Respect other contributors.
- Work together to improve the project.

Thank you for contributing to Smart Bank Management!
